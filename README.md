# 📊 Controle de Peso - Weight Tracker App

Um aplicativo web completo e funcional para monitorar seu peso, medidas e progresso de saúde. 100% gratuito, sem anúncios, sem login, totalmente privado!

## ✨ Funcionalidades

- ✅ **Dashboard Completo**: Visualização geral da sua saúde
- ✅ **Gráficos Dinâmicos**: Evolução de peso e cintura (semana/mês/tudo)
- ✅ **Registro de Peso e Medidas**: Adicione peso, cintura e data
- ✅ **Cálculos Automáticos**: IMC, % de gordura corporal, risco cardiovascular
- ✅ **Saúde Metabólica**: RCE (Relação Cintura/Estatura), RFM (Relative Fat Mass)
- ✅ **Risco Abdominal**: Avaliação personalizada por sexo
- ✅ **Histórico Completo**: Visualize todos os seus registros
- ✅ **Editar e Apagar**: Gerencie seus registros facilmente
- ✅ **Metas Personalizadas**: Defina seu peso alvo
- ✅ **Perfil Configurável**: Idade, altura, sexo, etc.
- ✅ **Sequência de Dias**: Acompanhe sua consistência
- ✅ **Compartilhar no WhatsApp**: Compartilhe seu progresso
- ✅ **Backup/Exportação**: Exporte seus dados em JSON
- ✅ **Dark Mode**: Suporte a tema escuro
- ✅ **100% Responsivo**: Funciona perfeitamente no celular
- ✅ **PWA**: Instale como app no celular

## 🚀 Instalação Rápida (5 minutos)

Veja o arquivo **INSTALL.md** para guia passo a passo completo!

Resumo:
1. Crie repositório no GitHub
2. Faça upload de todos os arquivos
3. Ative GitHub Pages em Settings → Pages
4. Acesse: `https://seu-usuario.github.io/weight-tracker/`
