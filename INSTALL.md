# 🚀 Guia Rápido de Instalação - GitHub Pages

## ⚡ Instalação Rápida (5 minutos)

### 1️⃣ Criar Conta no GitHub (se não tiver)
- Acesse: https://github.com
- Clique em **"Sign up"**
- Siga as instruções

### 2️⃣ Criar Novo Repositório
1. Faça login no GitHub
2. Clique no **"+"** no canto superior direito
3. Selecione **"New repository"**
4. Preencha:
   - **Repository name**: `weight-tracker` (ou outro nome)
   - **Description**: "Meu app de controle de peso"
   - Deixe como **Public** (público)
   - ✅ Marque **"Add a README file"**
5. Clique em **"Create repository"**

### 3️⃣ Fazer Upload dos Arquivos
1. Na página do seu repositório, clique em **"Add file"** → **"Upload files"**
2. Arraste TODOS os arquivos que você baixou:
   ```
   ✅ index.html
   ✅ profile.html
   ✅ add-entry.html
   ✅ app.js
   ✅ storage.js
   ✅ calculations.js
   ✅ manifest.json
   ✅ .gitignore
   ✅ README.md
   ```
3. Na caixa de texto embaixo, escreva: "Adicionando arquivos do app"
4. Clique em **"Commit changes"**

### 4️⃣ Ativar GitHub Pages
1. No seu repositório, clique em **"Settings"** (Configurações)
2. No menu lateral esquerdo, procure e clique em **"Pages"**
3. Em **"Source"**, selecione:
   - Branch: **main**
   - Folder: **/ (root)**
4. Clique em **"Save"**
5. Aguarde alguns segundos e atualize a página

### 5️⃣ Acessar Seu App! 🎉
Seu app estará disponível em:
```
https://SEU-USUARIO.github.io/weight-tracker/
```

*Substitua `SEU-USUARIO` pelo seu nome de usuário do GitHub*

**Exemplo**: Se seu usuário é `joaosilva`, o link será:
```
https://joaosilva.github.io/weight-tracker/
```

## 📱 Adicionar ao Celular (Atalho na Tela Inicial)

### iPhone/iPad:
1. Abra o link do seu app no **Safari**
2. Toque no botão de **Compartilhar** (quadrado com seta para cima)
3. Role para baixo e toque em **"Adicionar à Tela de Início"**
4. Toque em **"Adicionar"**
5. Pronto! O app aparecerá como um ícone na sua tela inicial

### Android:
1. Abra o link do seu app no **Chrome**
2. Toque no menu (**⋮**) no canto superior direito
3. Toque em **"Adicionar à tela inicial"** ou **"Instalar app"**
4. Toque em **"Adicionar"**
5. Pronto! O app aparecerá como um ícone na sua tela inicial

## ❓ Problemas Comuns

### O link não funciona
- Aguarde 5-10 minutos após ativar o GitHub Pages
- Verifique se todos os arquivos foram enviados
- Confira se o nome do repositório está correto no link

### Os dados não salvam
- Certifique-se de que não está usando modo anônimo/privado
- Permita cookies e armazenamento local no navegador
- Tente usar um navegador diferente (Chrome, Safari, Firefox)

### O app não aparece bonito no celular
- Certifique-se de ter adicionado o arquivo `manifest.json`
- Limpe o cache do navegador
- Tente adicionar à tela inicial novamente

## 🔄 Atualizações Futuras

Para atualizar o app:
1. Vá no seu repositório no GitHub
2. Clique no arquivo que quer atualizar
3. Clique no ícone de lápis (✏️) para editar
4. Faça as alterações
5. Clique em **"Commit changes"**
6. Aguarde alguns minutos
7. Limpe o cache do navegador no celular
8. Pronto! As mudanças aparecerão
